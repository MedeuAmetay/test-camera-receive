Выбор режима получения событий

Выберите, где хранить полученные события Hikvision: локально в браузере или на сервере с сохранением файлов.

UI / Local

Локально в браузере

Страница /ui получает поток и сохраняет его в IndexedDB только у тебя в браузере.

Подходит для быстрого просмотра, без записи данных на сервер.

Открыть UI
Capture / Server

События на сервер

Страница /capture включает серверный capture и хранит события и файлы на диске сервера.

Подходит для длительного сбора, анализа и выгрузки сохранённых файлов.

Открыть Capture
Camera setup

Настройка отправки с камер

Страница /camera/config отправляет массовые PUT-запросы на камеры с Digest Auth.

Подходит для первичной настройки notification endpoint и проверки ответов камер.

Открыть Camera setup
Setup upload type

Camera ARM

Страница /camera-arm массово отправляет настройку mixed target в JSON формате.

Подходит для быстрого включения/выключения mixed detection и binary upload.

Открыть Camera ARM
